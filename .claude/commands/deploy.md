description: Полный цикл деплоя с проверкой до и после
---
Прогони цикл: проверь пакет правок сам (build + по визуальным правкам — скрин) → commit с осмысленным сообщением → push → дождись деплоя (gh run list / скрипт ожидания) → проверь на проде → отчёт: ✅ что выкатил + ⚠️ design-решения, которые стоит обсудить.
