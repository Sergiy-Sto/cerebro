# 📦 Сопроводительный текст для раскатки kit в другой проект

> Использование: дай Claude в сессии другого проекта приложенный `claude-kit-starter.zip` + текст ниже (от «Обновление…» до конца).

---

Обновление claude-kit (версия 2026-06-18). К сообщению приложен claude-kit-starter.zip — эталон кита. Обнови ТОЛЬКО generic-файлы, не тронув специфичное для этого проекта.

⚠️ НЕ перезаписывай (в архиве — пустые шаблоны, твоё ценнее):
• ПРОЕКТ.md, ЖУРНАЛ.html — состояние проекта
• .claude/hooks/guard-files.js — твои защищённые пути
• задачи в TODO.md (но структуру мигрируй — см. ниже)

✅ Замени из архива (generic):
• CLAUDE.md
• ХУКИ И СКРИПТЫ - АРХИТЕКТУРА.md (если у тебя старое имя АРХИТЕКТУРА.md — переименуй через git mv)
• .claude/hooks/*.js КРОМЕ guard-files.js (включая новый anti-push-question.js)
• .claude/commands/*, .claude/settings.json
• scripts/test-*.sh

🔄 Мигрируй TODO.md (ОБЯЗАТЕЛЬНО — иначе todo-drain сломается):
переименуй секции в канбан «## 🟢 В РАБОТЕ / ## 🟧 ОЖИДАЕТ РЕШЕНИЯ ЗАКАЗЧИКА / ## 🟣 ПОЗЖЕ / ОТЛОЖЕНО / ## 🧊 ИДЕИ», легенду-справку убери вниз под «## ℹ️». Формат и легенду возьми из шаблона TODO.md в архиве, свои задачи перенеси в новые секции. Старые имена (Сейчас/Дальше/Заблокировано) хук больше НЕ ищет.

🗑 Удали УСТАНОВКА.md (если есть).

📦 Распаковывай архив через PowerShell Expand-Archive, НЕ через unzip (unzip корёжит кириллические имена файлов).

После применения:
1) перезапусти сессию (хуки грузятся на старте);
2) активируй Bypass permissions (Ctrl+Shift+M → 5);
3) самотест: rm -rf /несуществующая-папка-тест → ждём «ЗАБЛОКИРОВАНО ХУКОМ»;
4) прогони: bash scripts/test-guard-bash.sh, test-visual-check.sh, test-subagent.sh, test-anti-push.sh — все PASS;
5) запиши в ЖУРНАЛ: «kit обновлён до 2026-06-18».

Что нового: compound-команды разрешены, Playwright MCP для моб-скринов (провижининг в арх-доке), моб-нудж + фикс упавшего скрина, канбан-TODO, хук anti-push-question.
