История решений

2026-05-23 · Инициализация
Цветовая дифференциация блоков в интерфейсе
Решение: Визуально разделять три колонки Workbench (Stages / Cards / Card Detail) цветом фона или акцентной полосой — не просто серые блоки.
Почему: Пользователь явно попросил дифференцировать блоки цветом для улучшения навигации. При минималистичном UI без красивостей — цвет остаётся главным инструментом зонирования.
2026-05-23 · Инициализация
Создана структура управления памятью (CLAUDE.md / ЖУРНАЛ / ПРОЕКТ)
Решение: Три файла с разными ролями: CLAUDE.md — правила и стек (не меняется), ЖУРНАЛ.html — живое состояние (обновляется каждую сессию), ПРОЕКТ.html — справочник продукта.
Почему: Решает проблему потери контекста при сжатии памяти Claude Code. Claude.md автоматически подгружается при каждом старте.
